# Módulo Custom QR Invoice - Odoo 17

Este módulo ha sido desarrollado como parte del examen técnico para desarrollador Odoo.

## Funcionalidades Implementadas
- **Punto de Venta**: Agregada columna "Idioma" en el listado de clientes.
- **Facturación**: 
  - Generación de Código QR en facturas impresas.
  - Campos automáticos de Serie y Correlativo.
  - Campo "Canal de Ventas" con carga de datos inicial.
  - Reemplazo de fecha de factura por "Fecha de Emisión" con hora.
  - Pestaña de "Transferencias Relacionadas" vinculadas al pedido de origen.
